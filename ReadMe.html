Установка

Для установки чата, скопируйте все папки и файлы на сервер. После чего установите права файлам chmod 777. Зарегистрируйте свой ник. Зайдите в админ панель, используя логин Myst и пароль 123. В пункте профайл измените статус зарегистрированного вами ника на Администратор, затем в пункте управление пропишите настройки и исправьте логин главного на свой. Если неотображается защитная картинка при регистрации: находим на компе файл arial.ttf и копируем его в папку чата img.
